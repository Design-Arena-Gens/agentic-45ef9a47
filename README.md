# Make.com Pinterest Automation Designer

Design and document Pinterest automation scenarios tailored for the **us2.make.com** region. The app assembles
Pinterest triggers, downstream actions, QA guidance, and implementation steps so your automation team can deploy
reliable scenarios fast.

## Getting Started

```bash
npm install
npm run dev
```

Visit `http://localhost:3000` to start composing automations.

## Deploy

```bash
npm run build
npm start
```

Or deploy to Vercel (production):

```bash
vercel deploy --prod --yes --token $VERCEL_TOKEN --name agentic-45ef9a47
```
